# reverseip

***A Python Based Reverse IP Script To Find The Domains On The Server***

### Requirements

- Python (3.*)
- Python module `requests`

### Install modules

	pip install -r requirements.txt
	
### Tested on

- Windows 7/8/8.1
- Windows 10
